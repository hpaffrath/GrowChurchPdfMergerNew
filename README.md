# Microsoft Azure - Need to change Client Secret every 2 years
- Go into https://portal.azure.com/ and log in with Microsoft Account
- Search for App Registrations then click on "All Applications" tab
- Select GrowChurchPDFMerger
- Select "Client credentials" and there will be 1 secret
- Select "New client secret" to create a new secret


# Users
Potentially, anyone with a microsoft account can use the app.
